# CodeBhasha: The Hinglish Coding Assistant 🇮🇳

**Track:** [Student Track] AI for Learning & Developer Productivity

## 🚀 Overview
CodeBhasha is an AI-powered coding environment designed to democratize computer science education in India. It allows students to write, speak, and understand code in **Hinglish** (a blend of Hindi and English), removing the language barrier that stops many creative minds from learning logic and syntax.

## 💡 Key Features
- **🗣️ Bol Ke Code (Voice-to-Code):** Speak your logic in Hindi/Hinglish, and the AI converts it into executable Python code.
- **🛠️ Desi Debugger:** Instead of cryptic errors, get explanations in plain Hinglish (e.g., *"Bhai, tumne bracket band nahi kiya"*).
- **⚡ Real-Time Execution:** Run Python code directly in the browser.
- **📚 Smart Suggestions:** AI-driven autocomplete that understands intent, not just syntax.

## 🛠️ Tech Stack
- **Frontend:** React.js / Next.js
- **AI/LLM:** Amazon Bedrock (Claude 3.5 Sonnet)
- **Voice Services:** Amazon Transcribe (Speech-to-Text)
- **Deployment:** Vercel / AWS Amplify

## 🎯 Problem Statement
In India, millions of students study in vernacular mediums. When they transition to coding, they face a double barrier: learning **Logic** AND learning **English Syntax**. CodeBhasha bridges this gap by letting them focus on logic first, using the language they think in.
